## ⚙️ 重要設定

### Lookup Visits (查詢門檻)

```
建議值：100-200
├─ 100-200: 日常使用 ✅
├─ 300-500: 複雜局面
└─ 1000+: 專業分析
```

### Compute Visits (計算強度)

```
依裝置效能調整：
├─ 旗艦手機: 100-150
├─ 一般手機: 50-80 ✅
├─ 入門手機: 20-30
└─ 平板/電腦: 150-200
```

### 貼目 (Komi)

```
├─ 7.5: 正式比賽（推薦）✅
├─ 6.5: 部分舊規則
└─ 0.5: 讓子棋
```

## ❓ 常見問題速解

### Q: AI 說黑棋只有 48% 勝率？
**A:** 正常！貼目 7.5 的關係。

### Q: 分析太慢？
**A:** 降低 Compute Visits 到 30-50。

### Q: 能離線用嗎？
**A:** 能！內建開局庫 + 本地 AI。

### Q: 如何儲存棋譜？
**A:** 選單 ☰ → 棋局記錄 → 匯出 SGF。

## 🎯 Pro 小技巧

1. **開局用 Book**：前 10 手通常在開局庫，秒回！
2. **調整滑桿**：複雜局面提高 Compute Visits
3. **使用悔棋**：嘗試不同下法，比較勝率
4. **查看歷史**：點擊歷史紀錄頁籤，回顧每一手
5. **雲端同步**：啟用後自動備份，永不遺失
